# ElevateLabs - Responsive Landing Page

A simple responsive landing page built using HTML and CSS as part of the Web Development Internship Task 1.

# What I built

A clean landing page with:
- Header with logo and navigation links
- Hero section with heading, subtext, and a CTA button
- Footer with social links
- Mobile responsiveness using media queries

# Tech used

- HTML5
- CSS3 (Flexbox for layout)
- No frameworks or libraries

# How to run

Just open `index.html` in your browser. 

# What I learned

- Structuring a page with semantic HTML
- Using Flexbox for header and layout alignment
- Writing media queries for mobile screens
- Keeping CSS organized and readable
